(18) What are the media types allowed by CSS?

->CSS2 introduced @media, used to define different style rules for different media types.

->CSS3 introduced Media Queries used to check height and width of the viewport, resolution, height and width of the device.

->CSS3 media types like:-

  • all
    • used for all media types devices
  • print
    • used for printers
  • screen
    • Used for computer screens, tablets, smart-phones
  • speech
    • Used for screenreaders that "reads" the page
