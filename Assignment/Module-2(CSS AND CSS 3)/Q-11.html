(11) How to center block elements using css1?

->Block elements which are displayed at the starting of the line of code.

->By default it takes up the whole width of the content.

->Block elements like <div>, <li>, <ul>, <p>, <form>, etc.

->Method like ;

*By default width of block elements is 100% of the screen then by using margin for alignment of the block element.

*Another method like using center tag we can align the element.


->With the help of Margin, block elements are to be centered in css1.

->Margin-left: auto and Margin-right:auto our block elements will be centerd easily.
