Module-4 JAVASCRIPT

1)What is Javascript?


->Javascript is an scripting language.

->The programs in this language is called scripts.

->Javascript is an open and cross-platform language.

->In Javascript names, variables, keywords, and functions are case-sensitive.

->The browser interprets Javascript code.

->Javascript variables contain any value type.

->The same variable can have a string value, an integer value, etc.

->Variable can be declare with the use of keyword like var, const, and let.
