(17) What is the meaning of css Selector?

->CSS Selectors are used to select the html elements in which we want to style.

->There are many css selectors like Adjacent selector, Group selector, Universal selector, Pseudo class, Pseudo element,Child selector, General selector.

Adjacent selector

->It selects the element that are used first in div tag.
->for ex.
div + p {
background-color: yellow;
}

Group selector

->In this we can select more than one/two elements, to give same style.
->In group selector each selector is seperated by comma.
->for ex.
h2, p, i{
text-transform: capitalize;
}

Universal selector

->In this we can give Universally style to the all html elements.
->for ex.
*{
margin: 0;
padding: 0;
}

Pseudo class

->It is used to style an visited link, unvisited link differently.
->It is also used to give style an element when it is get focused.
->Pseudo class selectors like:- :active, :visited, :focus, :first-child, :last-child, :nth-child, even[:nth-child(2n)], odd[:nth-child(2n-1)], etc.
->for ex.
a:focus{
background-color:pink;
}

Pseudo element

->In this '::' is used for the selectors.
->It is used to give style to an first letter, first-line of an element.
->Also used to add content before or after the selected element.
->for ex.
p :: {
content : '';
color : red;
font-size : 15px;
}

Child selector

->It selects all the child elements with a particular parent element.
->for ex.
div > p {
font-size: 20px;
font-style: italic;
}

General selector

->It selects all the elements that are next siblings to the div tag elements.
->for ex.
div ~ p {
background-color: violet;
}
->ID Selector use id as an attribute to select an specific element.

->CLASS Selector selects html elements with a specific class attribute.
