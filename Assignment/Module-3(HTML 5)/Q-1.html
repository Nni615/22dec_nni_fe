(1) What are the new tags added in HTML5?

*Tags like;

(1)article:It represents the content of the document.

(2)aside:It represents the content that is slightly related to the page.

(3)audio:Defines an audio file.

(4)canvas:This is used for graphics such as graphs and games.

(5)cite:It will work same as italic tag. (6)datalist:Together with the new list attribute for the input.

(7)figure:It represents an flow of content and referenced as a single unit from the main flow of document.

(8)footer:It represents a footer section.

(9)hgroup:It represents header section.

(10)mark:It represents run of text in one document.

(11)meter:It represents a measurement.

(12)nav:It is used for navigation.

(13)output:It represents some type of output.

(14)progress:It represents a completion of a task.

(15)ruby:With ruby title and ruby paragraph allow for marking up ruby annotations.

(16)video:It defines video file.
