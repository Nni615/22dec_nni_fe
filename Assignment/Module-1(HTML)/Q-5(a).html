5(a).What are the different types of lists in html?

* There are three types of lists:-

#Ordered List

#Unordered List

#Description List

--> List which is specified in an sequence manner are known as Ordered list.

--> List which is specified in an non-sequence manner are known as Unordered list.

--> Description list contains tags like <dl> ,<dt> , <dd>.

*dl stands for Description-list.(defines Description-list)

*dt stands for Description-title.(defines Description-title)

*dd stands for Description-data.(defines Description-data)
