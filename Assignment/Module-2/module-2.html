MODULE-2(CSS and CSS 3)



(1) What are the benefits of using CSS?

->Using CSS we can design our webpage or website.

->Here, CSS can be apply in three ways like; EXTERNAL, INTERNAL, and INLINE.

->Using CSS we can able to set colors, fonts, space between paragraphs,images etc. in very short code.

->For using css "style"attribute is used in INLINE CSS, while for EXTERNAL and INTERNAL style tag is used.

(2) What are the disadvantages of CSS?

->CSS which works with one browser not always work or support with another browser, hence for that we use (webkit).

->Confusion beigns du to the many CSS levels like; CSS2, CSS3.

->CSS works differently on different browsers.

(3) What is the difference between CSS2 and CSS3?

->CSS3 supports more browsers than CSS2.
->With CSS2 only web safe fonts can be used, while in CSS3 special fonts such as Google fonts are used.
->CSS2 dosen't support the Border-box property, CSS3 supports Border-box property.
->CSS3 introduces several new properties with their values


(4) Name a few css style components.

->Components of css are:-

*Selector: In this we can set element name, id, and class name.

*Property: In this attribute like font-size, background-color, text-align, border, etc. are used.

*Value: In this we write the value of that property.

Selector{Property:value;}

x{color:pink;}
(5) What do you understand by CSS opacity?

->Opacity means transparency.

->Transparency of any of the color or image.

->It contains value i.e. 0 to 1.

->For ex. Opacity:0.3.

(6) How can the background color of an element be changed?

->Using CSS property i.e. background-color.

->Also we can use background:linear-gradient and radial gradient.

->In linear and radial gradient we can set their values like colors and degree like;

*background:linear-gradient(360 deg red green yellow blue);

(7) How can image repetition of the backup be controlled?

->We can set image repetition with the help of the property i.e. background-repeat.

->To controlled an image repetition we can use background-repeat:no-repeat, property.

->In result image will be visible once only.

(8) What is the use of background-position property?

->With the help of this background-position property we can able to set the position of an image.

->In this values can be set in pixels, left, right, top, bottom, center, percentage with including two sides or four sides.

*body{ background:url();

background-repeat:no-repeat;

background-position:50% 50%;

}

(9) Which property controls the image scroll in the background?

->background-attachment property is used to control the image scroll in the background.

->This property contains values like: Scoll and Fixed.

->By default their value is scroll, but for controlling the image to be Scrolled fixed value is used.

*body{ background:url();

background-attachment:fixed;

}

(10) Why should background and color be used as seperte properties?

->Background property used in two ways, one for background-color and another for image.

->While color property used only for text color.

->The background property is a complex property in css, and if it is combined with color, the complexity will increase.

(11) How to center block elements using css1?

->With the help of Margin, block elements are to be centered in css1.

->Margin-left: auto and Margin-right:auto our block elements will be centerd easily.

(12) How to maintain css specification?

->W3C (World Wide Web Consortium) maintains the css specification.

->The CSS specifications are the set of standards that defines how CSS should be used to style document.

(13) What are the ways to integrate css as a webpage?

->We can integrate css in three different ways; Inline css, Intenal css and external css.

->In Inline css we can add attribute "style" with the start tag.

->In Internal css after the tag using css is used.

->In external css we have to make seperte file of css and then link with the parent one file.

(14) What is embedded style sheets?

->An embedded style sheet allows to define the styles to the whole html document at one place.

-> Embedded style sheets refer to when we embed style sheet information into an html document using the style element.

(15) What are external style sheets?

->It is the type of css.

->In this we have to make an seperate file and then using the link tag we can link that seperated file within the haed tag.

-> That seperated file can be saved with .css.

*ul li{

list-style:none;

}

(16) What are the advantages and disadvantages of external style sheets?

->It provides the facility of changing the whole website by make changes in just one file.

->External css is more efficient method for styling the large website.

->Uploading or linking to multiple css files can increase your site's download time.

->External css requires the browser to download html and css files seperately.

(17) What is the meaning of css Selector?

->CSS Selectors are used to select the html elements in which we want to style.

->CSS Selector divides into two "id" and "class".

->ID Selector use id as an attribute to select an specific element.

->CLASS Selector selects html elements with a specific class attribute.

(18) What are the media types allowed by CSS?

->Media types that are allowed in css are like; all, aural, braille, embossed, handheld, print, projection, screen, tty, tv.

(19) What is the rule set?

->A CSS rule set that contains one or more selectors and one or more declarations.

*h1{

text-align:"center";

font-family:"sans-serif";

}
->here, h1 considered as an selector while declarations are text-align and font-family.

->In this more than one declartion is used along with their values.
