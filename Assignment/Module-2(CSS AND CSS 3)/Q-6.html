(6) How can the background color of an element be changed?

->Using CSS property i.e. background-color.

->Background-color can be used in many ways like;
  • Normally (by giving color name)
  • RGB (R-red G-green B-blue)
  • RGBA (R-red G-green B-blue A-alpha value)
  • HSL (H-Hue S-Saturate L-Lightness)
  • Hexa Code (#000)
->For ex.

<style>
div{
background-color:hsl(8, 100%, 70%);
}
</style>

->Also we can use background:linear-gradient and radial gradient.
