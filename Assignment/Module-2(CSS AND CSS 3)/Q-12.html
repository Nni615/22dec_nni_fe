(12) How to maintain css specification?

->W3C (World Wide Web Consortium) maintains the css specification.

->The CSS specifications are the set of standards that defines how CSS should be used to style document.

->The specification defines how CSS properties should be implemented by the browser.

->The specification includes:-
  • Syntax and datatypes of the languages
  • explanaition on CSS selectors
  • how we can assign values to properties
