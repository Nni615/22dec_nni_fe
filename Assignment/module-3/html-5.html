MODULE-3(HTML5)

(1) What are the new tags added in HTML5?

*Tags like;

(1)article:It represents the content of the document.

(2)aside:It represents the content that is slightly related to the page.

(3)audio:Defines an audio file.

(4)canvas:This is used for graphics such as graphs and games.

(5)command:It represents a command of the user.

(6)datalist:Together with the new list attribute for the input.

(7)details:It represents additional information.

(8)embed:It defines external content.

(9)figure:It represents an flow of content and referenced as a single unit from the main flow of document.

(10)footer:It represents a footer section.

(11)hgroup:It represents header section.

(12)keygen:It represents control for key pair generation.

(13)mark:It represents run of text in one document.

(14)meter:It represents a measurement.

(15)nav:It is used for navigation.

(16)output:It represents some type of output.

(17)progress:It represents a completion of a task.

(18)ruby:With ruby title and ruby paragraph allow for marking up ruby annotations.

(19)video:It defines video file.

(20)wbr:It represents a line break.
