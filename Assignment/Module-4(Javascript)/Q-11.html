11)What are all the looping structures in Javascript?


->Loop means line-of-code run repeatedly.

->Loop are of two types:

  • Entry Controlled loop
    • Condition will be checked first then the code will execute.

  • Exit Controlled loop
    • Code will execute first then the condition will be checked.

    ->Javascript contains loops like for loop, do..while loop, while loop, foreach loop, for-in loop, for-of loop.

    ->All the loop have different structures and ways to execute the condition of the loop.

    for loop

  • It is an entry controlled loop.
  • Inside the for loop there is declaration, condition, increament.
    • syn:-
    for(expression 1; expression 2; expression 3)
    {
       code to be executed;
    }

    do..while

  • It is an exit controlled loop.
  • In this first the code will execute then then the condition will be checked.
    • syn:-
    do
    {
       code to be executed
    }
    while(condition);

    while

  • It is an entry controlled loop
    • syn:-
    while(condition)
    {
       code to be executed;
    }

    forEach

  • The forEach() method calls a function for each element of an array.
    • syn:-
    array.forEach(function(){
    code to be executed
    });

    for-in

  • It is an entry controlled loop.
  • The code inside the loop is executed once for each property.
    • syn:-
    for(x in object)
    {
       code to be executed;
    }
