9.How we can club two or more rows or columns into a single row or column in an html table?

-->In to the HTML for clubing the rows and columns in an table "ROWSPAN" and "COLSPAN" is used as an attribute.

-->For clubing two or more rows into an single line "ROWSPAN" attribute is used.

-->For clubing two or more rows into an single line "COLSPAN" attribute is used.

For ex.

ROWSPAN
