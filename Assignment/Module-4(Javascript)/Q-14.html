14)What are all the types of popup boxes available in Javascript?



->There are three types of popup boxes available in Javascript.
  • Alert
  • Confirm
  • Prompt

    • ->Alert:- It will show simple message with ok button,

    ->Confirm:- It is used to get the confirmation message from the user side with ok and cancel button.

    ->Prompt:- In this user has to define the value inside the input box ith ok and cancel button.
